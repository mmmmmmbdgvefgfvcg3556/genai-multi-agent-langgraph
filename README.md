## genai-multi-agent-langgraph
Multi-Agent GenAI system using LangChain and LangGraph
## Requirements

The `requirements.txt` file contains all the Python dependencies required to run this project.
It ensures a consistent and reproducible environment setup.

### Dependencies
- **langchain** – for building LLM-powered agents
- **langgraph** – for orchestrating multi-agent workflows
- **openai** – for interacting with OpenAI language models
- **streamlit** – for building an optional user interface
- **tavily-python** – for enabling search and tool integrations

### Installation
Install all dependencies using:
```bash
pip install -r requirements.txt
